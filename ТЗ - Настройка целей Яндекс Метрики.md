# ТЗ - Настройка целей в Яндекс Метрике

**Проект:** streetwave® (street-wave.ru)
**Счётчик Метрики:** 98917979
**Дата:** 04.04.2026

---

## Задача

Создать цели в Яндекс Метрике (Настройка → Цели). Все цели типа **JavaScript-событие**. Идентификаторы целей уже вшиты в код сайта и срабатывают автоматически.

## Как создавать

1. Зайти в Метрику → Настройка → Цели → Добавить цель
2. Тип цели: **JavaScript-событие**
3. В поле "Идентификатор цели" вписать значение из столбца "Идентификатор цели" (например `form_custom_order`)
4. В поле "Название" вписать значение из столбца "Название в Метрике"
5. Сохранить

Повторить для всех 20 целей.

## Рекомендации по Яндекс Директу

Для оптимизации рекламных кампаний рекомендуется использовать следующие цели:

**Основные конверсии (для стратегии "Оптимизация конверсий"):**
- `form_custom_order` - главная цель B2C
- `form_brand_project` - главная цель B2B
- `payment_success` - покупка в магазине

**Микроконверсии (для ретаргетинга и анализа):**
- `visit_place_order` - посетил страницу заказа, но не отправил форму
- `open_modal_project` / `open_modal_event` - открыл форму, но не отправил
- `add_to_cart` - добавил в корзину, но не оплатил
- `click_telegram` / `click_whatsapp` - обратился напрямую

## Проверка

После создания целей и деплоя сайта:
1. Открыть сайт street-wave.ru
2. Открыть Метрику → Настройка → Цели → справа "Проверка целей"
3. Выполнить действие на сайте (например, перейти на /place-order)
4. В проверке должна загореться соответствующая цель
